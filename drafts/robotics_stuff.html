Setup

Connect uArm

Give temporary rights access uArm

  • $ sudo chmod 666 /dev/ttyACM0

Or get permanent USB permission to access uArm

$ cd /etc/udev/rules.d

Creat a file ttyUSB.rules and put the following line: KERNEL=="ttyUSB*", MODE="0666". Save the file and reconnect uArm USB to make it effective. (if you already have the permission to access USB, you can skip this step)

If your uArm is connected to ttyACM instead of ttyUSB (you can check using dmesg -c command after connecting your uArm), Creat a file ttyACM.rules and put the following line: KERNEL=="ttyACM*", MODE="0666". Save the file and reconnect uArm USB to make it effective. (if you already have the permission to access USB, you can skip this step) taken from here
